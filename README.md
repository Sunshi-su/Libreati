требования к коду:
1. управление стрелками (+ можно при перелючении цвет надписи меняется для наглядности)
2. предыдущие действия стираются
3. разные действия/функции/ для админа и гостя
4. пользователь при регистрации вводит фио и придумывает пароль -> все сохраняется в тхт файл
5. данные админа изначально записаны в системе /коде/ для входа
6. + файл тхт для хранения названия книг и их аннотации -> будет выводится при выборе книги. (пользователь читает и если не нравится возращаемся к меню выбора других книг)
7. при взятии книги регистрируем (на имя надо/не надо как хотите) дату / С (сегодня) ПО (+30дн)
Пример: "<название книги> период чтения(ну или как нибудь по-другому): 12.12.25-11.01.26 (включительно)"
